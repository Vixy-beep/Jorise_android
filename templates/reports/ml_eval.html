Key Findings
In-Sample Performance
  • All 6 RF models achieve 99.9–100% F1 on their training CSV
  • Near-zero false positive rate (<0.03%)
  • Confirms pipeline correctness and feature quality
Cross-File Generalization
  • All models collapse to F1 ≈ 0% on different attack types
  • 60% accuracy = predicting BENIGN for everything
  • DDoS model cannot detect WebAttacks and vice versa
Root Cause & Fix

Each model was trained on one attack type per CSV. DDoS flows (short duration, uniform packet length) are statistically distinct from PortScan flows (single-packet SYN, no response). The Random Forest memorized per-day fingerprints.

Fix: Train ONE unified model on ALL 8 CSVs concatenated (multi-class, using all attack labels, not binary). This is the retrain strategy.
